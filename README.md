# LC_MyDapper
1、.net core项目引入Dapper相关三方组件 2、Dapper简单CURD操作和批量URD操作 3、整合asp.net core mvc和Dapper、Autofac实现依赖注入和AOP，同时结合Redis实现缓存 4、通过Nginx实现集群负载均衡 5、数据库实现主从复杂，且从库读取实现负载均衡
